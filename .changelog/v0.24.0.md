# v0.24.0

- You can now use keyboard shortcuts to quickly navigate and take actions without reaching for the mouse
- Press **Ctrl+K** to open a searchable command bar where you can find and run any action by typing
- Press **?** to see a full list of all available shortcuts
- Use number keys **1–6** to jump between pages in the sidebar
- Press **N** to create a new item, **/** to search, and **F** to filter on the customers page
- The "Keyboard shortcuts" button in your account menu now opens the shortcuts guide
