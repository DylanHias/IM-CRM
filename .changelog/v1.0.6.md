# v1.0.6

## Fixes

- **Force re-sync no longer fails with 401** — the sync button in the admin panel was using the wrong login token (Microsoft account token instead of Dynamics 365 token), causing all D365 calls to be rejected. It now correctly fetches a D365-scoped token before syncing.
- **Re-authentication after session expiry now works correctly** — if your session expired and you had to log in again, the app was returning the wrong token type for D365 sync. Fixed so the correct D365 token is always used after re-login.
