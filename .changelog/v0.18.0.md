# v0.18.0

- Added a **Console** tab in the Admin panel that shows all application logs and errors in real-time — no need to open developer tools anymore.
- You can search through logs, filter by severity (info, warning, error), and clear the list.
- The console automatically scrolls to the latest entry and pauses when you scroll up to read older entries.
- Improved the consistency of all internal log messages so searching works reliably.
- Fixed several places where errors were silently ignored — they now show up in the console.
