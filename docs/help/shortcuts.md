# Keyboard Shortcuts

IM-CRM has keyboard shortcuts for common actions so you can work faster without reaching for the mouse.

## Command Palette

Press **Ctrl+K** (or **Cmd+K** on Mac) to open the Command Palette. This is a quick search bar where you can type any action or page name to jump to it instantly. Use the arrow keys to navigate results and Enter to select.

## Navigation

These shortcuts let you jump between pages. The number keys follow your sidebar order, so if you've reordered the sidebar in Settings, the numbers match your custom order.

- **1** — Go to first sidebar item (default: Customers)
- **2** — Go to second sidebar item (default: Sync)
- **3** — Go to third sidebar item (default: Follow-Ups)
- **4** — Go to fourth sidebar item (default: Opportunities)
- **5** — Go to fifth sidebar item
- **6** — Go to sixth sidebar item
- **S** — Open Settings
- **B** — Toggle sidebar collapse/expand

## General

- **Ctrl+K** / **Cmd+K** — Open Command Palette
- **?** — Open Keyboard Shortcuts guide
- **Esc** — Close any open dialog or modal

## Page Actions

These work on pages that support them:

- **N** — Create a new item (the type depends on which page or tab you're on)
- **/** — Focus the search bar
- **F** — Toggle the filter panel

## List Navigation

On the Customers page:

- **Arrow Down** — Move to the next customer in the list
- **Arrow Up** — Move to the previous customer
- **Enter** — Open the selected customer

## Data

- **Ctrl+Shift+R** — Trigger a sync with Dynamics 365

## Customizing Shortcuts

You can remap any shortcut in **Settings > Keybindings**:

1. Go to **Settings > Keybindings**
2. Find the shortcut you want to change
3. Click on it
4. Press the new key combination you want to use
5. The app will warn you if there's a conflict with an existing shortcut

To restore a shortcut to its default, click the reset button next to it. To restore all shortcuts, click **Reset All**.
