# Keyboard Shortcuts

Ingram Micro CRM has keyboard shortcuts for common actions so you can work faster without reaching for the mouse.

## Command Palette

Press **Ctrl+K** (or **Cmd+K** on Mac) to open the Command Palette. This is a quick search bar where you can type any action or page name to jump to it instantly. Use the arrow keys to navigate results and Enter to select.

## Customizing Shortcuts

You can remap any shortcut in **Settings > Keybindings**:

1. Go to **Settings > Keybindings**
2. Find the shortcut you want to change
3. Click on it
4. Press the new key combination you want to use
5. The app will warn you if there's a conflict with an existing shortcut

To restore a shortcut to its default, click the reset button next to it. To restore all shortcuts, click **Reset All**.
